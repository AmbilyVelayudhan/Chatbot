Mental Health Chatbot
A conversational AI-powered chatbot designed to support individuals dealing with depression. The chatbot provides a safe space for users to express their feelings, offers emotional support, and provides helpful resources.

🌟 Features
Natural Language Processing (NLP) – Understands and responds empathetically.
Sentiment Analysis – Detects user emotions and tailors responses accordingly.
Multilingual Support – Communicates in Malayalam, English, and Arabic.
Resource Suggestions – Provides mental health resources and self-help techniques.
Emergency Support – Guides users to professional help when necessary.
🛠️ Technologies Used
Python – Core programming language.
NLTK / spaCy – NLP processing.
Transformers (Hugging Face) – Advanced AI-driven conversations.
Streamlit / Flask – Web-based chatbot interface.
Firebase / MongoDB – User interaction data storage.
🚀 Installation & Setup
Prerequisites

Python 3.8+
